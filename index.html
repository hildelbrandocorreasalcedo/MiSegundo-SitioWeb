Mis Lenguajes De Programacion

Programar viene a ser el proceso de crear un software fiable mediante la escritura, prueba, depuración, compilación o interpretación, y mantenimiento del código fuente de dicho programa informático.

C++

Este lenguaje apareció a mediados de 1980, diseñado por Bjarne Stroustrup , es un lenguaje de programación compilado, extendido, estandarizado con paradigma: orientado a objetos, imperativo, programación genérica. Por que en los años 1900 la programación orientado a objetos ya había ganado terreno, por esas cuestiones se reinvento C++, actualmente esta en la versión C++14. Entre los lenguajes de programacion mas usados ocupa el puesto n.º 5

LEER MAS

C#

También llamado C Sharp es un lenguaje interpretado, totalmente orientado a objetos desarrollado por Microsoft bajo el paradigma de estructurado, imperativo, orientado a objetos, dirigido por eventos, funcional, genérico, reflexivo, apareció en el año 2000 la primera versión, que fue liderado por Anders Hejisberg. El lenguaje ocupa el puesto N.º 5 entre los mas populares.

LEER MAS

JAVA

Java un lenguaje de programación de propósito general influido por Pascal, C++, Objective-C , apareció en el año 1995, fue desarrollado por James Gosling & Sun Microsystems, entre sus paradigmas mas sobresalientes es; escribir el programa una vez y que se ejecute en cualquier dispositivo y sistema operativo, es un lenguaje de alto nivel totalmente orientado a objetos.

LEER MAS

JavaScript

En la actualidad este lenguajes es ampliamente usado en el desarrollo de aplicaciones web y de escritorio, es un lenguaje de programación interpretado del lado del cliente con paradigma, Programación funcional, Programación basada en prototipos, imperativo, Interpretado (Scripting), apareció en el año 1995 diseñado por Brendan Eich.
